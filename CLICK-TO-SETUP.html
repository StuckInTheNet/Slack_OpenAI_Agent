Step 1: Install Node.js (if you don't have it)

Node.js is what runs the bot. It's free and safe.

Download Node.js

Download it, install it, then come back here.

Test if it worked:

Windows
Mac

1. Press Windows Key + R

2. Type powershell and press Enter

3. Type this and press Enter:

node --version

You should see something like v18.17.0

1. Press Cmd + Space

2. Type terminal and press Enter

3. Type this and press Enter:

node --version

You should see something like v18.17.0

Step 2: Run the Setup Wizard

Now we'll run the interactive setup that walks you through everything.

Windows
Mac

Method 1 (Easiest):

1. Hold Shift and right-click in this folder

2. Click "Open PowerShell window here"

3. Copy and paste this command:

node setup-wizard.js

Method 2 (If that doesn't work):

1. Press Windows Key + R

2. Type powershell and press Enter

3. Type cd (with a space), then drag this folder into the window

4. Press Enter, then type:

node setup-wizard.js

Method 1 (Easiest):

1. Press Cmd + Space

2. Type terminal and press Enter

3. Type cd (with a space), then drag this folder into Terminal

4. Press Enter, then type:

node setup-wizard.js

Step 4: Customize Your Bot (Optional)

Want to give your bot a custom name and personality?

Change the name:

  1. Go back to api.slack.com/apps
  2. Select your bot app
  3. Click "App Home" in the sidebar
  4. Change "Display Name" (e.g., "Team AI Assistant")
  5. Upload a custom avatar if you want
  6. Click "Save"
